Електронавантажувач STILL RX 20-14 C (1.4 т, 2019 р.в.)

Ціна: 205 000 грн

Опис
Продається професійно відновлений електричний навантажувач STILL RX 20-14 C 2019 року випуску. Навантажувач пройшов повне технічне обслуговування та має нову перевірку UVV (перевірка безпеки). Ідеальний для роботи на складах, має компактні розміри та високу надійність. Повністю готовий до роботи.

Характеристики

Модель: STILL RX 20-14 C

Рік випуску: 2019

Вантажопідйомність: до 1 400 кг

Висота підйому: 3 160 мм

Тип щогли: DUPLEX

Напрацювання: 13 517 годин

Стан: Професійно відновлений, нова перевірка UVV.

Переваги: Надійний, якісно відновлений, готова сервісна підтримка.
